# Privacy Policy **Last updated: February 10, 2026** ## Hide Gmail Spam Count - Chrome Extension ### Overview Hide Gmail Spam Count ("the Extension") is developed by Pebble Tools. This privacy policy explains how we handle user data. ### Data Collection **We do not collect any personal data.** The Extension: - Does NOT collect personal information - Does NOT track browsing history - Does NOT transmit any data to external servers - Does NOT use analytics or tracking tools - Does NOT store any data outside your browser ### Data Storage The Extension stores only your preferences (such as which labels to hide) locally in your browser using Chrome's built-in storage API (`chrome.storage.local`). This data: - Never leaves your device - Is not accessible to us or any third party - Is deleted when you uninstall the Extension ### Permissions The Extension requires the following permissions: - **Host permission (mail.google.com)**: Required to modify Gmail's user interface and hide badge counts.
